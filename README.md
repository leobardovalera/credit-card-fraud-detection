# Credit Card Fraud Detection

End-to-end data science portfolio project for detecting fraudulent credit card transactions using Excel, SQL, Python, machine learning, Tableau, cloud technologies, and GitHub.

---

## Overview

This project aims to build an end-to-end data science pipeline for detecting fraudulent credit card transactions.

The project serves both as a learning experience and as a public portfolio demonstrating practical skills in:

- Excel
- SQL
- Python
- Statistics
- Machine Learning
- Tableau
- Cloud Computing
- Git and GitHub

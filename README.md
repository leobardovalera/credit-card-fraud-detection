# Credit Card Fraud Detection

## Overview

This project aims to build an end-to-end data science pipeline for detecting fraudulent credit card transactions.

The project is intended both as a learning experience and as a public portfolio demonstrating practical skills in:

- Excel
- SQL
- Python
- Statistics
- Machine Learning
- Tableau
- Cloud Computing
- Git and GitHub

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- SQL
- Tableau
- AWS
- GitHub
